# README #

Installation
------------

1. Clone this repository wherever you want

...<b>$ git clone https://github.com/Jbuchs/vhost.git</b>

2. Move the script in /usr/bin

3. Give execution rights on the script

...<b>$ chmod + x vhost</b>

4. Call the script from the terminal as root

...<b>$ sudo vhost</b>
